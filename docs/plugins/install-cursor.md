# Install Mantle Forge — Cursor

## One line

```bash
npm install && npm run plugin:cursor
```

Reinstall:

```bash
npm run plugin:cursor
```

Then **Developer → Reload Window** in Cursor.

## Verify

- Settings → Rules → skills `mantle-*` visible under Agent Decides
- Command `/flagship-workflow` available in chat

## Use

```txt
/flagship-workflow
```

Or paste the flagship prompt from [README.md](./README.md).

Docs: [Cursor plugins](https://cursor.com/docs/plugins)
