# Report artifact checklist

For a complete Mantle project report, keep these artifacts available:

- `reports/test-summary.md` or captured `npx hardhat test` output
- `reports/gas.md`
- `reports/security.md`
- `deployments/mantleSepolia.json` (or equivalent deployment artifact)
- explorer links and transaction hashes from the deployment artifact

If the test section is missing from the final report, it usually means no test summary artifact was provided to the report generator.
