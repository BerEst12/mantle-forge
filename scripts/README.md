# Scripts

| Script | Purpose |
|--------|---------|
| `installer/install-plugin.js` | Install plugin per runtime |
| `installer/verify-plugin.js` | Verify bundle + CLI smoke tests |
| `plugins/mantle-forge/scripts/verify-env.js` | Session hook — check `MANTLE_FORGE_ROOT` |

Run from repo root:

```bash
npm run plugin:verify
npm run plugin:sync-skills
```
